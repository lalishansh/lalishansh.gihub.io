Introducing blacklily

blacklily is a minimal Jekyll theme curated for storytellers. It is designed and developed by myself for a friend who writes short stories.

blacklily features

  • Compatible with GitHub Pages
  • Responsive design (looks just as good on mobile)
  • Syntax highlighting, with the help of Pygments
  • Markdown and HTML text formatting
  • Pagination of posts
  • Sticky posts
  • Tags
  • Excerpt management
  • Disqus comments
